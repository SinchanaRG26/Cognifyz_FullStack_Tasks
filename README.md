## 🚀 Task 2: Inline Styles, Basic Interaction, and Server-Side Validation
### 📸 Demo
()

### ✨ Features
-Extended registration form (Name, Email, Password, Age)
-Inline CSS styling for improved UI
-Client-side validation using JavaScript
-Password length validation (Minimum 6 characters)
-Age validation (18+ required)
-Server-side validation using Express.js
-Temporary server-side storage using JavaScript array
-Dynamic user count display after successful registration
-Proper form submission handling using POST method

### 🛠️ Technologies Used
- HTML5 – Structure of the web page
-CSS3 – Inline styling
-JavaScript – Client-side validation
-Node.js – Backend runtime
-Express.js – Server framework
-EJS – Server-side rendering template engine
